★（星マーク）は主催者・管理者おすすめの演芸会です。すでにみんなに公開されている情報なので、自分のメモとして「★行く」など状態を付けながら使えます。

使い方：メモを並び替え・絞り込み

  1. 上部の 並び基準 で「開催日」または「入力日」を選択。
  2. 並び順 で「降順／昇順」を切り替え。
  3. 予約済みのみ にチェックを入れると、予約済の演芸会だけに絞れます。
  4. 行く予定 から「行く／行かない／未定」で絞り込み。

使い方：演芸会を登録する（2通り）

  1. 方法A：チラシから自動登録。「チラシを取り込む」ボタンを押し、手元のチラシ画像をアップロードすると、AIが日時・会場・出演者を読み取り、修正フォームに自動入力します。最後に内容を確認して保存。
  2. 方法B：右上の「＋ 新規登録」ボタンからフォームに直接入力する。
  3. 登録後はリストに自分の思い出として並びます。タイトルをクリックすると詳細・編集画面へ移動できます。
チラシ取り込みを試したときの実体験メモ：
  • PNG / JPEG だけでなく PDF をそのまま投げてもOK。解析は10〜20秒ほどかかります（ボタンが「解析中…」表示に変わります）。
  • 会名・日付・開場／開演時間・会場名・住所・電話番号・出演者は、ほぼ正確に自動抽出されました。
  • 予約メールアドレスは「予約メール」欄ではなく「備考」欄に入ることがあります。気になる場合は手動で予約メール欄に貼り直してください。
  • U25割など複数の料金区分がある場合、当日金額／前売金額の両方に大きい方の金額が入ることがあります。前売金額を「U25：1000円」などに直すと正確になります。
  • 「曲師」など出演者扱いの人物も 出演者リストにまとめて入ります（例：沢村理緒）。
  • 抽出後はすべての欄を一度目で見て、必ず「保存する」を押す前に微調整するのがおすすめです。

各項目について

  1. アイコン：プロフィール画像。タップすると画像をアップロードできます。
  2. 表示名：他ユーザーから見える名前。空欄ならユーザーIDが表示されます。
  3. メールアドレス：ログインID と通知の宛先になります。
  4. 予約情報（氏名・電話番号）：会場へ予約メールを送るとき、サイトが自動で本文に差し込んでくれる連絡先です。先に入れておくと予約が早くなります。
  5. 好きなジャンル：落語／講談／浪曲／その他から複数選択可。今後のおすすめ表示に使われます。
  6. 好きな演者：最大5名まで登録可能。
  7. 編集後は必ず最下部の「保存する」ボタンを押してください。

登録する
  1. ユーザー名（半角英数字・記号）を決める。後でログインID／URLの一部として使われます。表示名は別途プロフィールから設定できるので、ここはシンプルなIDでOK。

セキュリティのヒント：パスワードは8文字以上に加え、英数字＋記号の組み合わせを推奨。パスワード管理アプリ（1Password / Bitwarden / Mac標準のキーチェーンなど）に保存しておくと安心です。

STEP 6：ログイン後、まずプロフィールを少し設定

登録直後はアバターも空、表示名もユーザーIDのままです。右上アバター → 「プロフィール」から、最低限の3つだけでも整えておくと使いやすくなります。

  1. 表示名 ＝ 自分の本名や好きなニックネームを入力。
  2. 好きなジャンル ＝ 落語／講談／浪曲／その他から1つ以上チェック。
  3. 会場予約をメールで取りたい場合は 「予約情報」の氏名・電話番号 も入れておく。
  4. 下部の 「保存する」 をクリック。
